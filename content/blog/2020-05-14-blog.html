While R is a program commonly used in data science and statistics, python can also be used to accomplish similar purposes. Various data manipulations can be done on both programs. One manipulation that can be done on both programs is clustering. Clustering is important in data science because it create groupings of the data points. This creates a visualization of the positioning and correlation of the data.

Both R and python can create clusters, with differences in functions and names. In the example below, clustering is completed using R on the iris dataset using kmeans.

In python, the kmeans function allows for clustering and assignment of the data points into three clusters, similar to what was done in R. In python, the clusters are (0,1,2), which is similar to the clusters in R (1,2,3). Python can also calculate the cluster means using .cluster_centers_. It is possible to run python code while using R by the reticulate package (). This allows Rstudio to import python packages and run python codes alongside R codes. Base on the example below, python was also able to create three clusters and calculate the cluster means.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn import datasets
from sklearn.cluster import KMeans

iris = datasets.load_iris()
x=iris.data
y=iris.target

kmeans = KMeans(n_clusters=3)
y_kmeans = kmeans.fit_predict(x)
print(y_kmeans)
